# Chiara & Jonas — Seating Plan

Wedding seating plan webapp for 06.06.2026 at Villa Rubini Redaelli, Dongo, Lake Como.

## How to update guest data

Edit `index.html` — the `DATA` array at the top of the `<script>` block contains all tables and guests. Each guest has:
- `s` = seat label (shown on floor plan)  
- `f` = full name (shown in search bar)

## Hosting

This is deployed via GitHub Pages. Any push to `main` goes live automatically.
